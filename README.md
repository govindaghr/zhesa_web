# dzongkha

#### to run on local host, 
```
git clone https://github.com/govindaghr/zhesa_web.git
pipenv shell
pip install -r requirements.txt

python manage.py runserver 3000
```
#### requirements:
```
django==3.2.9
django-crispy-forms
crispy-bootstrap5
djangorestframework
django-filter
gunicorn
whitenoise
markdown
```
